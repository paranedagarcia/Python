A continuación se presenta una propuesta de **curso completo de nivel profesional**, orientado a personas que ya poseen conocimientos básicos de Python y análisis de datos. El curso está diseñado con un enfoque eminentemente práctico, utilizando datos reales y las principales bibliotecas del ecosistema Python.

La estructura no solo enseña a construir gráficos, sino que desarrolla la capacidad de **contar historias con datos (Data Storytelling)**, seleccionar la visualización correcta para cada problema analítico y comunicar resultados a audiencias técnicas y ejecutivas.

---

# Curso: Visualización de Datos para el Análisis y la Comunicación de Resultados con Python

## Duración sugerida

**40 horas**

* 16 horas teóricas
* 20 horas de laboratorio
* 4 horas proyecto final

---

# Objetivo General

Desarrollar las competencias necesarias para diseñar, construir e interpretar visualizaciones de datos utilizando Python, aplicando principios de percepción visual, comunicación efectiva y análisis exploratorio para transformar datos en información útil para la toma de decisiones.

---

# Objetivos Específicos

Al finalizar el curso los participantes serán capaces de:

* Comprender los principios de percepción visual y diseño de información.
* Seleccionar el tipo de gráfico más adecuado según la naturaleza de los datos y el objetivo analítico.
* Utilizar las principales bibliotecas de visualización de Python.
* Construir gráficos estadísticos de calidad profesional.
* Desarrollar dashboards interactivos.
* Aplicar principios de Data Storytelling.
* Comunicar resultados de forma efectiva para audiencias técnicas y no técnicas.
* Diseñar informes visuales reproducibles.

# Actividades de Aprendizaje

Durante el curso se desarrollarán diversas actividades orientadas a consolidar los conocimientos teóricos mediante la práctica:

* Exposiciones sincrónicas dialogadas sobre principios de percepción visual, comunicación de resultados y análisis exploratorio.
* Demostraciones prácticas del uso de bibliotecas de visualización en Python.
* Talleres guiados para construir gráficos estadísticos, visualizaciones interactivas y mapas.
* Laboratorios de análisis exploratorio utilizando conjuntos de datos reales de distintos dominios (salud, economía, transporte, medio ambiente y sector público).
* Ejercicios de análisis crítico de visualizaciones, identificando fortalezas, debilidades y sesgos en gráficos publicados.
* Actividades de rediseño de visualizaciones para mejorar la claridad y efectividad comunicacional.
* Desarrollo progresivo de un portafolio personal de visualizaciones.
* Proyecto final integrador orientado a comunicar hallazgos mediante una narrativa visual coherente.

## Resultados de Aprendizaje

Al finalizar el curso, los participantes serán capaces de:

* Diseñar visualizaciones alineadas con objetivos analíticos y comunicacionales.
* Aplicar principios de percepción visual y diseño de información para mejorar la comprensión de los datos.
* Construir gráficos estadísticos e interactivos de calidad profesional utilizando el ecosistema de Python.
* Integrar visualizaciones en flujos de análisis exploratorio de datos (EDA) para identificar patrones, tendencias y anomalías.
* Elaborar dashboards interactivos que faciliten el monitoreo de indicadores y la exploración de información.
* Comunicar hallazgos de manera efectiva mediante técnicas de *Data Storytelling*, adaptando el mensaje a diferentes tipos de audiencias.
* Generar informes reproducibles y visualmente consistentes que apoyen la toma de decisiones basada en evidencia.

Esta propuesta proporciona una cobertura integral que va desde los fundamentos de la percepción visual hasta el desarrollo de dashboards y narrativas con datos, combinando competencias técnicas en Python con habilidades de comunicación analítica, lo que la hace especialmente adecuada para profesionales del análisis de datos, ciencia de datos y gestión pública o empresarial.
